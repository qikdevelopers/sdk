(static) channel(key)

Source:
  • qik.socket.js, line 63
Create an event dispatcher instance and subscribe to a new socket channel. This can then be used to receive socket events for this channel
Example
const socketChannel = await sdk.socket.channel('awesome:notifications')
socketChannel.addEventListener('message', function(message) {
   console.log('Received a new message from the socket')
})

// Disconnect from this channel and remove all listeners
socketChannel.destroy();
Parameters:
Name Type Description
key String The key or name of the channel to subscribe to
